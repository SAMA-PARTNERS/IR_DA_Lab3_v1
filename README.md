# IR-DA-03 — Offline Log Triage & Timeline Reconstruction (ORYXYBER)

This is a single-workstation investigation lab. You receive a small case pack of exported Linux artifacts and must determine:
- where the access came from,
- which account was compromised,
- what happened immediately after access, and
- whether persistence was established.
